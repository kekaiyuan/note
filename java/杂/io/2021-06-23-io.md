# NIO

NIO 中一切都是异步的，但是网络传输是需要时间的。
如果不调用 sync() 方法，那么程序会发出一个网络请求就会直接往下执行。
如果需要等待得到网络回复，就调用 sync() 方法，它会同步阻塞地等待网络
