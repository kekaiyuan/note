Github 如何写脚注

# 如何在Github中使用脚注？
## md标准格式

```
文章内容[^1]

[^1]: 脚注1的说明
```
文章内容[^1]

[^1]:脚注1的说明

## 自定义锚点
```
文章内容<sup id="a1">[[1]](#f1)</sup>

<span id="f1">1[.^](#a1)</span> 脚注1的说明
```
文章内容<sup id="a1">[[1]](#f1)</sup>

<span id="f1">[1.^](#a1)</span> 脚注1的说明

**注意：**"["与"^"之间必须隔开，可以是空格、小数点等一切字符
